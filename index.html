Free and Open Source Software Projects
  • selin : selin is a minimalist Lisp dialect. The selin interpreter is written as a C++ header library, so that it can be embedded easily into C++ projects.
  • gurowankenobi : gurowankenobi is a C++ interface for GLPK (GNU Linear Programming Kit). The C++ API of gurowankenobi is a compatible subset of the C++ API of the commercial solver Gurobi.
  • fef : fef (for-each-file) is a command-line tool for searching for and/or doing operations on multiple files/directories. It can be considered as an alternative to the classical 'find' command of unix-like systems.
  • sheldon.vim (co-developer: R.K. Srivastava) : sheldon.vim is a plugin for the Vim text editor. It is a command interpreter, inspired by the command-line operating system shells. It is intended to allow the user to do simple shell operations without having to leave Vim.
  • Akarsu Basic : Akarsu Basic is a dialect of BASIC programming language. The Akarsu Basic source codes are translated to JavaScript, so that the BASIC programs are run on Web browsers.
  • UfakPython : UfakPython is a tiny subset of Python programming language, designed for small embedded devices, like PICs. UfakPython source codes are translated to C, which can then be compiled by Small Device C Compiler for the target device.
